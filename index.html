My ToDo List

A simple todo app to boost your productivity.

View project onGitHub

Features

  • Adding a task
  • Editing a task
  • Star/Unstar a task
  • Duplicate task check
  • Mark a task as done + Remove it
  • Undo previous actions
  • Redo previous undos
