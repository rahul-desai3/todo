Todo Application
================

A simple todo application for boosting productivity. This application includes following features:

  - Adding a task
  - Editing a task
  - Star/Unstar a task
  - Duplicate task check
  - Mark a task as done + Remove it
  - Undo previous actions
  - Redo previous undos
